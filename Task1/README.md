## Number Guessing Game (Java)

## Overview

This is a simple Java console-based Number Guessing Game where the player tries to guess a randomly generated number between **1 and 100** within limited attempts.

The game also includes:

- Score tracking
- Hints for close guesses
- Replay option
- Final score display

## Features

- Random number generated between 1 and 100  
- Maximum 5 attempts per round  
- "Too High" / "Too Low" hints  
- "Very Close" hint if guess is near the correct number  
- Score increases by 10 for each correct guess  
- Option to play multiple rounds  
- Final score shown at end

## Technologies Used

- Java
- Random Class
- Scanner Class
- Loops
- Conditional Statements
