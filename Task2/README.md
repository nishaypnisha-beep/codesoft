# Student Marks Calculator (Java)

## Overview

This is a simple Java console application that calculates a student's:

- Total Marks
- Average Percentage
- Overall Percentage
- Grade

The program takes the student's name, number of subjects, and marks for each subject as input.

## Features

- Accepts student name
- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average percentage
- Calculates overall percentage
- Displays grade based on performance

| Average Percentage | Grade |
| ------------------ | ----- |
| 90 and above       | A     |
| 75 - 89            | B     |
| 60 - 74            | C     |
| 50 - 59            | D     |
| Below 50           | F     |


## Technologies Used

- Java
- Scanner Class
- Conditional Statements
- Loops
