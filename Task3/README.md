# ATM Interface System (Java)

## Overview

This is a simple Java console-based ATM Interface System that simulates basic banking operations.  
The project uses Object-Oriented Programming concepts by connecting an **ATM class** with a **BankAccount class**.

Users can perform the following operations:

- Check Account Balance
- Deposit Money
- Withdraw Money
- Exit the System

## Features

- Menu-driven ATM interface
- Check current account balance
- Deposit money into account
- Withdraw money from account
- Input validation for transactions
- Prevents withdrawal beyond available balance
- User-friendly messages
- Uses classes and objects (OOP)

## Technologies Used

- Java
- Scanner Class
- Classes and Objects
- Encapsulation
- Switch Case
- Loops
