# CodeSoft course Java

## Overview

This repository contains four beginner-friendly Java console projects developed to practice core programming concepts such as loops, conditions, arrays, object-oriented programming, and collections.

The projects included are:

1. Number Guessing Game
2. Student Grade Calculator
3. ATM Interface System
4. Student Management System


## Technologies Used

* Java
* Scanner Class
* Random Class
* ArrayList
* Object-Oriented Programming (OOP)
* Conditional Statements
* Loops
* Switch Case


## Project 1: Number Guessing Game

### Description

A fun console game where the user tries to guess a randomly generated number between 1 and 100 within limited attempts.

### Features

* Random number generation
* 5 attempts limit
* Too High / Too Low hints
* Very Close hint
* Score tracking
* Replay option


## Project 2: Student Grade Calculator

### Description

A Java program that calculates student results based on marks entered for multiple subjects.

### Features

* Enter student name
* Enter marks for subjects
* Calculate total marks
* Calculate average percentage
* Calculate overall percentage
* Assign grade


## Project 3: ATM Interface System

### Description

A console-based ATM simulation using Object-Oriented Programming concepts.

### Features

* Check account balance
* Deposit money
* Withdraw money
* Input validation
* Menu-driven system
* Uses BankAccount and ATM classes


## Project 4: Student Management System

### Description

A Java-based management system to store and manage student records.

### Features

* Add student
* Remove student
* Search student
* Display all students
* Uses ArrayList
* Menu-driven interface


## Concepts Used

* Variables and Data Types
* User Input Handling
* Loops
* Conditional Statements
* Arrays / Collections
* Methods
* Classes and Objects
* Encapsulation
* Constructor Usage


## Purpose

These projects were created to strengthen Java programming fundamentals and understand real-world problem solving using Java.
