## Student Management System

## Description

This is a Java-based Student Management System that allows users to manage student records through a simple console interface.
Users can add, remove, search, and display student details efficiently.

##Technologies Used

- Java
- ArrayList (Java Collections)
- OOP Concepts

## Features

- Add a new student
- Remove a student by roll number
- Search for a student
- Display all students
- Menu-driven console interface

## Project Structure

- "Main.java" → Handles user interaction
- "Student.java" → Defines student details
- "StudentManagementSystem.java" → Core logic for managing students

## Concepts Used

- Classes & Objects
- Encapsulation
- ArrayList
- Loops & Conditional Statements
